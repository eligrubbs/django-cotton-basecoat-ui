# Basecoat Design System

Your own copy of a `django-cotton` implementation of all Basecoat UI components.

## Usage

All components are under `templates/cotton/ds` so you can use them like `<c-ds.button>...</c-ds.button>`.

## Why a Django App?

The only reason this is a django app is so that it can ship by default with a demo site (once you configure your static assets).
