# django-cotton-basecoat-ui
Basecoat UI components powered by django-cotton

## TLDR; What is this repo?

This repository maintains a [django-cotton](https://django-cotton.com/) django app component library styled with [Basecoat UI](https://basecoatui.com/). 

It creates a django app called `django_basecoat` with the templates and a demo site to get you started seeing everything in action.

## Who is this repo for?

1. You use Django
2. You have chosen Basecoat UI as the basis for your websites design system.
3. You don't mind using `django-cotton` to manage some of your components.
